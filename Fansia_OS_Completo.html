⚡ Diccionario del Vibecoder

10 prompts universales. Copia y pega.

1. Generar código base

"Eres senior en [Python]. Genera un script que [descripción]. Incluye: logging, try/except, type hints, comentarios. Que sea ejecutable."

2. Depurar error

"Código: [pegarlo]. Error: [pegarlo]. Dame 3 causas probables, cómo verificarlas, código corregido."

3. Explicar código

"Explica este código como si tuviera 15 años. Qué hace cada bloque, por qué es necesario, qué pasa si lo quito."

4. Refactorizar

"Refactoriza: más legible, modular, funciones, type hints, PEP8. No cambies funcionalidad."

5. Añadir feature

"A este código quiero añadir [feature]. Solo muéstrame qué líneas añadir, dónde, qué imports."

6. Mejorar prompts

"Mejora este prompt: [mi prompt]. Hazlo más específico con restricciones y formato de salida."

7. Escribir tests

"Tests con pytest para este código: caso normal, borde, error. Usa assert y parametrize."

8-10. Comparar · Convertir · Seguridad

"Compara [lib A] vs [lib B] para [tarea]" · "Convierte de [lib A] a [lib B]" · "Revisa vulnerabilidades en este código"

▲ Plan de Contingencia

3 atajos para cuando te estanques.

⚡ Atajo 1: ML sin teoría

  • Solo memoriza: import → split → fit → predict → score
  • Random Forest funciona sin tuning.
  • ¿Accuracy baja? → más datos, no cambiar modelo.

⚡ Atajo 2: Error → IA

  • Copia el error a ChatGPT: "Explica como si tuviera 12 años"
  • VS Code: selecciona error → Ctrl+I → "Fix this"
  • Regla: máx 30 min por error. Pídele a la IA.

⚡ Atajo 3: LangChain manual

  • RAG con Python puro: openai.Embedding.create() + numpy para coseno
  • Fórmula: sim = np.dot(e1,e2)/(np.linalg.norm(e1)*np.linalg.norm(e2))

📉 Plan B por atraso

  • Sem 1-2 atrasada? Salta API, ve directo a ML
  • Sem 5-6 atrasada? Salta ChromaDB, usa FAISS
  • Nunca saltes el proyecto final. Hazlo aunque sea solo RAG.

🔄 Depuración en 15 min

  • 0-5 min: Lees el error, buscas typo
  • 5-10 min: IA: "Dime 3 causas"
  • 10-15 min: "Reescribe más simple"
  • >15 min: Borra, prompt diferente
