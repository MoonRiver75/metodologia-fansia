# Fansia OS — Metodología Fansia

Plan de estudio de 60 días para aprender IA desde cero con Vibecoding (70% AI-assisted, 30% conceptos). Incluye ruta de aprendizaje, checklist diario, calendario, diario de aprendizaje, recursos, prompts universales y plan de contingencia.

## 🚀 Despliegue

Este es un PWA (Progressive Web App) instalable. Abrir `index.html` en Chrome y usar "Add to Home Screen" para experiencia nativa.

## 📦 Archivos

- `index.html` — App principal (PWA entry point)
- `manifest.json` — PWA manifest
- `sw.js` — Service worker (cache offline)
- `icon-192.png` / `icon-512.png` — Iconos PWA

## 🔧 Uso

1. Abrir en Chrome mobile o desktop
2. Instalar como PWA (Chrome menu → Add to Home Screen)
3. Configurar perfil (nombre + icono)
4. Seguir la ruta de 8 semanas, marcar checklist, escribir diario

Hecho con ❤️ y vibecoding.
